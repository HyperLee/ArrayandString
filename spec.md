# Array / String

在電腦科學中，陣列 (Array) 與 字串 (String) 是最基礎且密切相關的資料結構。簡單來說，字串通常被視為一種「字元的陣列」。

核心定義與差異陣列 (Array)：一種在連續記憶體空間中儲存相同類型元素的集合。特性：可透過索引 (Index) 直接存取元素，長度通常在初始化時固定。多維性：支援一維、二維甚至多維結構。字串 (String)：一串字元的序列，用於表示文字。底層結構：在 C 語言中，字串是以 \0 (Null character) 結尾的字元陣列。不可變性：在 Java 或 Python 等語言中，字串通常是不可變的 (Immutable)，而陣列則是可變的 (Mutable)。
